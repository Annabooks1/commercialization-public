# Update Surface (WoW64)

<conditional_block> <conditions> <docset value="standalone"></docset> </conditions>

>[!NOTE]
You can find the latest version of this test documentation on MSDN at the following link:

-   <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/3398d420-5f93-4613-a282-a5374a825ea9">http://msdn.microsoft.com/en-us/library/windows/hardware/3398d420-5f93-4613-a282-a5374a825ea9</b>


</conditional_block>

This automated test verifies correct implementation of the driver functionality that underlies the **IDirect3DDevice9::UpdateSurface** method.

The test uses the **IDirect3D9::CheckDeviceFormat** method to determine the combinations of formats and surface types that the driver supports.

For all valid surface-type combinations, the test performs a comprehensive set of **UpdateSurface** operations. These operations include full surface and subrect copies, with and without translation.

The test accomplishes automatic verification by rendering both the destination surface and a reference surface and comparing the rendered images. The test uses the **LoadSurfaceFromSurface** function in Microsoft® Direct3D® 9 to prepare reference images.

This topic applies to the following test jobs:

-   Update Surface

-   Update Surface (WoW64)
