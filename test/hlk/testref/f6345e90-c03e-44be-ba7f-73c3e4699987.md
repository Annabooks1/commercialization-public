# Windows Client System Check For WDDM 1.2 Display Driver Support

<conditional_block> <conditions> <docset value="standalone"></docset> </conditions>

>[!NOTE]
You can find the latest version of this test documentation on MSDN at the following link:

-   <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/888d22d9-bd7a-4f30-81fb-038dc5e2b104">http://msdn.microsoft.com/en-us/library/windows/hardware/888d22d9-bd7a-4f30-81fb-038dc5e2b104</b>


</conditional_block> This test verifies that all display adapters/chipsets in the system are using display drivers that are WDDM 1.2 version or greater.
