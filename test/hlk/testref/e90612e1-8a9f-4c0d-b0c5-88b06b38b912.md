>[!NOTE]
You can find the latest version of this test documentation on MSDN at the following link:

-   <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/c6a8c430-2e2e-408b-a4d9-11db6c2b0848">http://msdn.microsoft.com/en-us/library/windows/hardware/c6a8c430-2e2e-408b-a4d9-11db6c2b0848</b>


</conditional_block> The Document Property Test exercises the printer driver's implementation of the DocumentProperty API. The test calls the API with various data and then validates the returned values.
