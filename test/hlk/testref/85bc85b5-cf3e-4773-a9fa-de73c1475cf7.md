# <span id="p-hlk-test.85bc85b5-cf3e-4773-a9fa-de73c1475cf7"></span>WDDM PnPStop Test


This automated test disables and enables the display adapter.

This test disables the display adapter and then enables it again. Setupapi is used for disabling and enabling the display device. This is an alternate method of enabling and disabling the display adapter to using Device Manager.
