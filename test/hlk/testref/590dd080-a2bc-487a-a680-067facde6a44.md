# <span id="p-hlk-test.590dd080-a2bc-487a-a680-067facde6a44"></span>Camera Driver Usage Indicator Manual Test


This test validates that the camera indicates its streaming status properly. When a camera is streaming, the LED (or on screen indicator) should be on. When the Camera is not streaming, the LED/indicator should be off
