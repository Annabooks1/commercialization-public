# WDDM BVT2 (WoW64)

<conditional_block> <conditions> <docset value="standalone"></docset> </conditions>

>[!NOTE]
You can find the latest version of this test documentation on MSDN at the following link:

-   <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/2153cc34-11b1-43d2-b3f1-b710998dc0fd">http://msdn.microsoft.com/en-us/library/windows/hardware/2153cc34-11b1-43d2-b3f1-b710998dc0fd</b>


</conditional_block>

Thisautomated test verifies system stability under the Windows® Display Driver Model (WDDM) and verifies the functionality of Windows Vista video drivers.

The test consists of a set of test groups. Each test group exercises one subset of the Windows Vista user-mode Device Driver Interface (DDI). The test does not use the Microsoft® Direct3D® runtime. Instead, it calls directly into the user-mode driver and the kernel-mode entry points in the operating system. Drivers must not crash or leak memory during the test.

This topic applies to the following test jobs:

-   WDDM BVT2

-   WDDM BVT2 (WoW64)
