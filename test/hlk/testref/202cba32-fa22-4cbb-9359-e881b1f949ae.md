# USB 3.0 Hub Enumeration Bounce

<conditional_block> <conditions> <docset value="standalone"></docset> </conditions>

>[!NOTE]
You can find the latest version of this test documentation on MSDN at the following link:

-   <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/8c5a7f57-9d90-4b54-95d3-945d08ee26cb">http://msdn.microsoft.com/en-us/library/windows/hardware/8c5a7f57-9d90-4b54-95d3-945d08ee26cb</b>


</conditional_block> This test verifies that when a USB 3.0 hub is connected to an xHCI port configured at USB 2.0 speeds, the hub reconnects on the SuperSpeed port when the SuperSpeed port is re-enabled and that the companion port stays connected.
