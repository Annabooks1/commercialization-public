# Test.MinMaxContacts

<conditional_block> <conditions> <docset value="standalone"></docset> </conditions>

>[!NOTE]
You can find the latest version of this test documentation on MSDN at the following link:

-   <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/64dbaf07-a31c-4bf8-b2da-f6feccff12d9">http://msdn.microsoft.com/en-us/library/windows/hardware/64dbaf07-a31c-4bf8-b2da-f6feccff12d9</b>


</conditional_block>

This test verifies that the device supports the minimum amount of contacts required (3) and does not support more than the maximum number of contacts (5).

## Running the test

Before you run the test, complete the test setup as described in the test requirements: <xref rid="p_hlk_test.mouse_or_other_pointing_device_testing_prerequisites">Mouse or other Pointing Device Testing Prerequisites</b>.

The tool automatically passes/fails the requirement based on how many contacts the device reports that it supports.
