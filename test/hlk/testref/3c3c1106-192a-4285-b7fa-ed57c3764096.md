# WDDM WSSections

<conditional_block> <conditions> <docset value="standalone"></docset> </conditions>

>[!NOTE]
You can find the latest version of this test documentation on MSDN at the following link:

-   <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/4cfc5e13-1f9e-40b7-8152-49112ec388a5">http://msdn.microsoft.com/en-us/library/windows/hardware/4cfc5e13-1f9e-40b7-8152-49112ec388a5</b>


</conditional_block>

This automated test verifies that graphic drivers are not wide opened to all processes.

Some video vendors might release drivers with shared portable executable (PE) sections that are wide opened to all processes. This test finds all installed display driver files and scans for write-shared sections. The test will fail if it finds a write-shared memory section.
