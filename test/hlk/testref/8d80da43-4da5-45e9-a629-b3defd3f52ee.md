# DXGI DirectFlip

<conditional_block> <conditions> <docset value="standalone"></docset> </conditions>

>[!NOTE]
You can find the latest version of this test documentation on MSDN at the following link:

-   <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/f3b5654f-37f4-4884-ba90-f5f8c3d85aa4">http://msdn.microsoft.com/en-us/library/windows/hardware/f3b5654f-37f4-4884-ba90-f5f8c3d85aa4</b>


</conditional_block>

This automated test verifies that a graphics adapter or chipset driver publishes the SupportDirectFlip field as TRUE in the DXGK\_DRIVERCAPS structure.

>[!NOTE]
This field is required for support of DirectFlip. DirectFlip enables a minimum amount of memory bandwidth for displaying full-screen content while ensuring smooth transitions between full-screen applications, other applications, and the desktop environment.
