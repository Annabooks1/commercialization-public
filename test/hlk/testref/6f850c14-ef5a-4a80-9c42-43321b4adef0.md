## Parameters

| Parameter name                                    | Parameter description                                                                                                   |
|---------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| <mark type="bullet_intro">Iterations</b>          | Number of iterations to perform.                                                                                        |
| <mark type="bullet_intro">ChargerOnPort1</b>      | Type of charger on port 1 (J2) of the USB Connection Exerciser: None, Usb2Pc, Usb3Pc, Dcp, TypeC, TypeCPd, NonCompliant |
| <mark type="bullet_intro">PdVoltageOnPort1</b>    | PD Voltage expected on port 1 (J2) of the USB Connection Exerciser (If Applicable)                                      |
| <mark type="bullet_intro">TypeCCurrentOnPort1</b> | Current supplied by charger on port 1 (J2) of the USB Connection Exerciser (Type-C & PD only)                           |
| <mark type="bullet_intro">ChargerOnPort2</b>      | Type of charger on port 2 (J3) of the USB Connection Exerciser: None, Usb2Pc, Usb3Pc, Dcp, TypeC, TypeCPd, NonCompliant |
| <mark type="bullet_intro">PdVoltageOnPort2</b>    | PD Voltage expected on port 2 (J3) of the USB Connection Exerciser (If Applicable)                                      |
| <mark type="bullet_intro">TypeCCurrentOnPort2</b> | Current supplied by charger on port 2 (J3) of the USB Connection Exerciser (Type-C & PD only)                           |
| <mark type="bullet_intro">ChargerOnPort3</b>      | Type of charger on port 3 (J6) of the USB Connection Exerciser: None, Usb2Pc, Usb3Pc, Dcp, NonCompliant                 |
