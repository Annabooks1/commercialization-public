## Parameters

| Parameter name                                  | Parameter description                                                                                                                                                                |
|-------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <mark type="bullet_intro">TestCycles</b>        | Number of cycles to run the test for                                                                                                                                                 |
| <mark type="bullet_intro">DoFullLogging</b>     | Enable flag for full logging of ETW traces in case of failure, and re-run this test.                                                                                                 |
| <mark type="bullet_intro">CopyLogsOnFailure</b> | Enable flag to copy ETW log traces to subfolder 'ETWlogs' in case of failure, and re-run this test. This also copies the failure logs to hlkx package to be shared for investigation |
