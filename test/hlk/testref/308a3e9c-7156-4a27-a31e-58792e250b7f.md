# <span id="p_hlk_test.308a3e9c-7156-4a27-a31e-58792e250b7f"></span>WDDM Process Callout


This automated test is a stress test that verifies system stability during process destruction.

This test creates many processes that do windowed and full-screen d3d9 rendering. The test then asynchronously terminates these processes. Drivers must not cause the system to stop responding (hang) or crash while running this test.
