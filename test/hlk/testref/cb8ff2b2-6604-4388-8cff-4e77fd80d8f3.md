# <span id="p-hlk-test.cb8ff2b2-6604-4388-8cff-4e77fd80d8f3"></span>WDDM Primary Surface


This automated test verifies that drivers correctly implement presentation operations.

This test exercises the kernel-mode presentation DDI in a variety of ways. Drivers must succeed present calls and not crash or fail to respond (hang) during this test. Specifically, the following scenarios:

-   Present-blts to 8bpp, 16bpp, and 32bpp surfaces

-   Color-fill presents to 8bpp, 16bpp, and 32bpp surfaces

-   Presents to staging buffers

-   Primary to primary present-blts

-   Primary to primary present-blts

This topic applies to the following test jobs:

-   WDDM Primary Surface

-   WDDM Primary Surface (WoW64)
