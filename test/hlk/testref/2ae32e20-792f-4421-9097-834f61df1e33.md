# <span id="p_hlk_test.2ae32e20-792f-4421-9097-834f61df1e33"></span>Camera Driver System Test - MediaCapture - ValidateAdvanceIsoSpeedControlValues


This test validates that that select values within the advertiesd range of Advanced ISO Speed values within range exposed by the camera work properly. The test will attempt to take a photo after setting the ISO value.
