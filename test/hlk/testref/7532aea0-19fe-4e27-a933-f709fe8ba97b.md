# USB Type-C UCM Data Role Swap

>[!NOTE]
You can find the latest version of this test documentation on MSDN at the following link:

-   <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/68867b5f-836a-49e6-9d5b-db369ba118cb">http://msdn.microsoft.com/en-us/library/windows/hardware/68867b5f-836a-49e6-9d5b-db369ba118cb</b>


</conditional_block>

## Parameters

| Parameter name                               | Parameter description                                                   |
|----------------------------------------------|-------------------------------------------------------------------------|
| <mark type="bullet_intro">SwapsToPerform</b> | Number of data role swaps to perform. The minimum is 2.                 |
