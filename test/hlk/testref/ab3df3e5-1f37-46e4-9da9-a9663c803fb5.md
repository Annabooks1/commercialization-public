## Running the test

Before you run the test, complete the test setup as described in the test requirements: <xref rid="p_hlk_test.wireless_lan__80211__testing_prerequisites">Wireless LAN (802.11) Testing Prerequisites</b>. Make sure that device has been enabled, drivers are properly installed, and the device is properly inserted.

The test runs the following test 5 times: <xref rid="p_hlk_test.24f9bc8e-591d-4fa0-8dfd-735369428379">Wlan L2 Test 02 - Connect to a WPA2 PSK AES AP (strict)</b>. Any error encountered in any of the five attempts will fail the test.
