## Parameters

| Parameter name                                     | Parameter description                                                                                                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <mark type="bullet_intro">SlotMappingIndexList</b> | Executor to slot mapping: The slot with i-th index will be mapped to Executor i. Format: \[index,index,...\] Example: 1,0 which means Slot 1 will be mapped to Executor 0, Slot 0 will be mapped to Executor 1. |
| <mark type="bullet_intro">SlotIndex</b>            | Specifies the 0-based slot index whose information is requested                                                                                                                                                 |
