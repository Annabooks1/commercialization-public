## Running the test

This test requires a secondary computer running a Nano Server configuration. The Nano Server computer and the HLK client computer should contain the same driver and device to be certified. An example configuration is shown below:

![example configuration for nano server capable testing of a device driver](../images/nano_config_example.jpg)

Before running this test, you must first load the Compatibility Playlist. Once loaded, you must then manually add the Device.DevFund.Server.Nano feature so that the necessary tests can be scheduled. For more information, see <xref rid="p_hlk.manually_add_features">Manually add features</b>.
