## Running the test

Before you run the test, complete the test setup as described in the test requirements: <xref rid="p_hlk_test.mouse_or_other_pointing_device_testing_prerequisites">Mouse or other Pointing Device Testing Prerequisites</b>.

The RA tool is required for this test. For more information, see <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/hh967413">How to Measure Touch Panning Latency</b>.

You will need to fit your RA tool with a smaller, adjustable PTP bracket.

Adjust the radius of the circle to 15mm and position the touchpad such that the contact will stay on the touchpad for the full circle.
