# <span id="p-hlk-test.90dbe89f-d76e-426c-a377-d54d6f077e9a"></span>D3D12 - Resource Binding - Graphics Root Constant Buffer Views


Test 3D Root Constant Buffer Views by binding 4 views and adding up the contents in a pixel shader and outputting to Render Target for verification.
