## Running the test

Before you run the test, complete the test setup as described in the test requirements: <xref rid="p_hlk_test.windows_touch_testing_prerequisites">Windows Touch Testing Prerequisites</b>.

Clean input is important throughout the system and across applications. Whether a user is performing a hold gesture to learn more about a user interface (UI) element, playing a game, or holding a key on the software keyboard, this fidelity allows software to correctly interpret a user's intention and provide an optimal touch experience.

This test presents twelve distinct scenarios that examine jitter during static contact, linear motion, and radial motion. Most of these interactions require the use of assistive technology.

In addition to validating that the jitter specification is respected, this test verifies that HID reports conform to the HID protocol that is described in <xref hlink="http://go.microsoft.com/fwlink/?LinkID=226808">Windows Pointer Device Data Delivery Protocol</b> and defined in the <xref hlink="http://go.microsoft.com/fwlink/?LinkID=286770">HID I2C v1.0 Protocol Specification</b>.
