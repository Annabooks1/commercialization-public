## Running the test

Before you run the test, complete the test setup as described in the test requirements: <xref rid="p_hlk_test.windows_touch_testing_prerequisites">Windows Touch Testing Prerequisites</b>.

This test can help to ensure that touch systems can effectively tolerate common environmental conditions.

This test presents two scenarios: the only distinction between the two scenarios is the power source. For the <mark type="term">Battery</b> portion of the test, you should remove the external power source from a mobile system.

In each case, the test validates that a contact that is held at the minimum required distance from the screen does not register any touch contacts. For precise values, see **Windows Hardware Certification Requirements: Devices**.
