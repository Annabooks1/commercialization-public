# Functional test for monitor container ID

<conditional_block> <conditions> <docset value="standalone"></docset> </conditions>

>[!NOTE]
You can find the latest version of this test documentation on MSDN at the following link:

-   <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/3ebf937d-f334-4f9a-ad31-bdfc00c621a3">http://msdn.microsoft.com/en-us/library/windows/hardware/3ebf937d-f334-4f9a-ad31-bdfc00c621a3</b>


</conditional_block>

This automated test applies SetupAPI functions to retrieve the monitor containerId and check the consistency of those after reboot. The ones achieved from monitor EDID and ELD info from audio devices are also used for verification.
