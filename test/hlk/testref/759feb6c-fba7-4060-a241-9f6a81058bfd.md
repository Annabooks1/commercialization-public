This test is only applicable to the hard disk that is configured as the boot drive.

Before you run the test, complete the test setup as described in the test requirements: <xref rid="p_hlk_test.hard_disk_drive_testing_prerequisites">Hard Disk Drive Testing Prerequisites</b>. This test requires 2 GB of unallocated disk space on the disk being tested.
