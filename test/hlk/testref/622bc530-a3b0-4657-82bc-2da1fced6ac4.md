You can find the latest version of this test documentation on MSDN at the following link:

</conditional_block> This runs the superfetch speed test on the drive. It will setup a volume and proceed to run the test 10 times. This test will determine whether the drive will be suitable for Windows to use as a hybrid drive. If it is too slow, then it wonâ€™t use the hybrid feature at all.
