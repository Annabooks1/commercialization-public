## Running the test

Before you run the test, complete the test setup as described in the test requirements: <xref rid="p_hlk_test.mouse_or_other_pointing_device_testing_prerequisites">Mouse or other Pointing Device Testing Prerequisites</b>.

To finish this test, see <xref hlink="http://msdn.microsoft.com/en-us/library/windows/hardware/dn195876">How to Measure Touch Down Latency by using an Acoustic Measurement Tool</b>.

Use pttouch.exe instead of the touch file.

Only 50 taps during the collection phase are required for PTP.
