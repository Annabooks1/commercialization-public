# Microsoft-Windows-TwinUI-Hide

The `Microsoft-Windows-TPM-Tasks-TwinUI-Hide` setting specifies whether to hide the link to an advanced settings app in the Pen and Windows Ink Settings page.


## Value


<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<tbody>
<tr class="odd">
<td><p><em>0</em></p></td>
<td><p>Show the link. This is the default value.</p></td>
</tr><tr class="even">
<td><p><em>1</em></p></td>
<td><p>Hide the link. </p></td>
</tr>
</tbody>
</table>
