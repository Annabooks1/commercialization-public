# What's new in customizations


This topic provides information about what's new in Windows customizations across all Windows 10 devices.

For more information about Windows customizations, see [Customize](customize-portal.md).

## What's new in customizations for Windows 10, version 1607


<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<thead>
<tr class="header">
<th>Item</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p>[ProvisioningCommands](p_icd_settings.provisioningcommands)</p></td>
<td><p>You can now leverage the ability to include scripts in a provisioning package to install Win32 applications. For instructions on how to do this, see [Install a Win32 app using a provisioning package](p_customize_converged.install_a_win32_app_using_a_provisioning_package).</p></td>
</tr>
<tr class="even">
<td><p>[Adaptive hibernate](p_customize_converged.adaptive_hibernate)</p></td>
<td><p>Adds support for triggers that allow the system to hibernate intelligently. These triggers eliminate resume to a dead battery and provides a great connected standby experience by ensuring that the system remains in CS for as long as possible.</p></td>
</tr>
<tr class="odd">
<td><p>Taiwan Alerts</p></td>
<td><p>You can now configure Taiwan Alerts as part of the [Emergency notifications](p_phCustomization.emergency_notifications) customization.</p></td>
</tr>
<tr class="even">
<td><p>[Dialer codes to launch diagnostic applications](p_phCustomization.dialer_codes_to_launch_diagnostic_applications)</p></td>
<td><p>This customization now contains functionality to support different app types and behaviors. Learn how this affects any existing configurations you may have or find out what you need to do to support a newer app type and behavior.</p></td>
</tr>
<tr class="odd">
<td><p>[Configure partner email account](p_customize_converged.partner_email_account)</p></td>
<td><p>Partners can add additional email account options to the <strong>Add an account</strong> list found on the <strong>Email &amp; app accounts</strong> screen in <strong>Settings</strong>. Beginning with Windows 10, version 1607, this customization is available for both mobile and desktop editions of Windows</p></td>
</tr>
<tr class="even">
<td><p>[Customize the taskbar](p_customize_desktop.customize_the_taskbar)</p></td>
<td><p>New. Starting in Windows 10, version 1607, you can pin up to three additional apps to the taskbar by adding a taskbar layout modification file, for example, TaskbarLayoutModification.xml. You can specify different taskbar configurations based on SKU, device locale, or region.</p></td>
</tr>
<tr class="odd">
<td><p>[Set dark theme](p_customize_desktop.set_dark_theme)</p></td>
<td><p>Windows 10, build 1607 exposes a new personalization setting for end users, allowing them to express preference whether to see applications which support the setting in a dark or light mode.</p></td>
</tr>
</tbody>
</table>
