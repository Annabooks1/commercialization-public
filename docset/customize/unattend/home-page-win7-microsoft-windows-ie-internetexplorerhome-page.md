# Home\_Page


`Home_Page` specifies the Uniform Resource Locator (URL) for the default home page that appears when the user starts Internet Explorer®. To set additional home pages, use the Microsoft-Windows-IE-InternetExplorer/[StartPages](startpages-win7-microsoft-windows-ie-internetexplorerstartpages.md) setting.

## Values


<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<tbody>
<tr class="odd">
<td><p><em>Url</em></p></td>
<td><p>Describes the fully qualified URL path. <em>Url</em> is a string.</p></td>
</tr>
</tbody>
</table>

 

This string type does not support empty elements. Do not create an empty value for this setting.
