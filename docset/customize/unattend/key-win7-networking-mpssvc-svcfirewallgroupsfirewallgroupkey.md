# Key


`Key` specifies the unique string to identify the name of the firewall group.

The value for `Key` is added to the answer file as an attribute of the [FirewallGroup](firewallgroup-win7-networking-mpssvc-svcfirewallgroupsfirewallgroup.md) element. The attribute `wcm:keyValue` is used to identify each unique firewall group. For example, you can specify three different firewall groups by using the `Key` value of **RemoteDesktop**, **WindowsMediaPlayer**, and **HomeGroup**.

## XML Example


The following XML output shows how to set Windows Firewall groups.

``` syntax
<FirewallGroups>
      <FirewallGroup wcm:action="add" wcm:keyValue="RemoteDesktop">
      <Active>true</Active> 
      <Group>Remote Desktop</Group> 
      <Profile>all</Profile> 
   </FirewallGroup>
</FirewallGroups>
```
