# IoT Core manufacturing


Windows 10 IoT Core (IoT Core) is a version of Windows 10 that is optimized for smaller devices with or without a display. IoT Core uses the rich, extensible Universal Windows Platform (UWP) API for building great solutions.

OEMs can use manufacture and deploy IoT Core using a variety of existing hardware. To see what's currently supported, see [Windows compatible hardware development boards](https://msdn.microsoft.com/library/windows/hardware/dn914597) and the [Hardware Compatibility List](http://go.microsoft.com/fwlink/?LinkID=532948).

When developing your own board, check out the [Minimum hardware requirements for IoT Core](https://msdn.microsoft.com/library/windows/hardware/dn915086#iotcore).
