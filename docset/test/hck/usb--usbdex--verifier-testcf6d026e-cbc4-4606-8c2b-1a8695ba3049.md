# USB (USBDEX) Verifier Test


This test verifies that a USB device and driver handles simulated failures while using the Microsoft® Extensible Host Controller Interface (xHCI) stack.

## Running the test


Before you run the test, complete the test setup as described in the test requirements: [USB Device.Connectivity Testing Prerequisites](usb-deviceconnectivity-testing-prerequisites.md).

For information included in the Windows Driver Kit on this test, see [USB client driver verifier (Windows Drivers)](http://msdn.microsoft.com/library/windows/hardware/jj672841.aspx).
