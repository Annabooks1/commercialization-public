# Diagnosability - Triage Logs - Clear SetupAPI logs


This optional test is provided to help troubleshoot certain test failures and is not required for certification.

This optional test can be used to delete existing SetupAPI logs from the **%windir%\\inf** folder on the Windows Hardware Certification Kit (Windows HCK) test machine. For more information about Setup API logs, see [SetupAPI Logging (Windows Vista and Later)](http://go.microsoft.com/fwlink/p/?linkid=303741).
