# DF - Non-Test - Uninstall EDT test filter driver (Optional)


You can use this optional test to remove the EDT filter driver from all targets on the system. This driver is heavily used as part of the Device Fundamentals tests.

This optional test is provided to help troubleshoot certain test failures and is not required for certification.
