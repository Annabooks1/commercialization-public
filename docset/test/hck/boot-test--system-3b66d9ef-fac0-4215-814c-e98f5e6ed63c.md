# Boot Test (SYSTEM)


This test determines whether the test client is booting from a particular device and if the device is configured correctly.

**Note**  
This is a system-specific implementation of an existing test. For more information on this test, see [Boot Test (LOGO)](boot-test--logo-1660c636-710c-462a-9920-7557fd55d230.md).
