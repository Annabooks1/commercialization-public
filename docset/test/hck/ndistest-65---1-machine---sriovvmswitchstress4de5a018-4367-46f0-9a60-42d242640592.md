# NDISTest 6.5 - 1 Machine - SRIOVVMSwitchStress


This test stresses the SRIOV capable miniport adapter in the presence of a vSwitch with VFs assigned. This is a single machine job and it needs additional configuration for execution. Please follow the same instructions as specified for the [NDISTest 6.5 - 1 Machine - SRIOVBasicVerification](ndistest-65---1-machine---sriovbasicverification616e3e15-2635-4810-b4bf-ec90eeb00a4f.md) test.
