# Audio Logo Test (System)


This test validates test cases for audio devices on generic streaming of audio.

**Note**  
This is a system-specific implementation of an existing test. For more information on this test, see [Audio Logo Test - Certification](audio-logo-test---certification91d27525-b5cf-4a7d-a52d-fdcf6a530571.md).
