# IGD Data Connection Test (Verify UPnP IGD is enabled)


This test adds port mappings to the router by using UPnP interface and then verifies those port mappings by sending traffic from a WAN machine to a LAN machine. The test fails if router does not properly implement UPnP port mapping interface.
