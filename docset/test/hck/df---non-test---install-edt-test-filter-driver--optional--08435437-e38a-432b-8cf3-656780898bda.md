# DF - Non-Test - Install EDT test filter driver (Optional)


You can use this optional test to install the EDT filter driver on the targets of your choosing. This driver is heavily used as part of the Device Fundamentals tests.

This test is provided to help troubleshoot certain test failures and is not required for certification.
