# NDISTest 6.5 - 2 Machine - SRIOVTestingRegistry


This test validates the SRIOV registry settings of the miniport adapter. This is a 2-machine test and is like any normal LAN certification test which requires Support and Message adapter renaming. It does not require any other additional configuration.
