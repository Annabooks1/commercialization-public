# WDDM CCD Test for Validation of Basic Topology (Multi Mon) - Multihead


This automated test verifies that Windows® supports the basic topologies of internal, external, extended, and clone. The test tries to validate and set these basic topologies if validation succeeds.

## Running the test


Before you run the test, complete the test setup as described in the test requirements: [Graphic Adapter or Chipset Testing Prerequisites](graphic-adapter-or-chipset-testing-prerequisites.md).

In addition, this test requires a video card or chipset that has a Windows Display Driver Model (WDDM) 1.1 driver.

**Note**  
If you run this test on a system without multiple monitors, you will be prompted to add a second monitor. If your device family does not support multimon, select "Continue" to force the test to run in single monitor configuration. Some tests can only be run on systems with multiple monitors, in which case the test will be skipped.
