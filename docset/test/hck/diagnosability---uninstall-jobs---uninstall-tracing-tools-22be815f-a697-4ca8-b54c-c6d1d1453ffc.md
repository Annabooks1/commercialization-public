# Diagnosability - Uninstall Jobs - Uninstall tracing tools


This optional test is provided to help troubleshoot certain test failures and is not required for certification.

This optional test can be used to uninstall [Tracefmt](http://go.microsoft.com/fwlink/p/?linkid=296461) from on the Windows Hardware Certification Kit (Windows HCK) test system.
