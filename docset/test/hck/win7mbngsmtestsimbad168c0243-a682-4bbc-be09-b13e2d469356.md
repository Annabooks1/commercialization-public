# Win7.MBN.GSM.TestSimBad


This test validates the return values of various control operations when a bad SIM is inserted.

## Running the test


Before you run the test, complete the test setup as described in the test requirements: [Mobile Broadband Testing Prerequisites](mobile-broadband-testing-prerequisites.md). The test requires a bad SIM to be inserted into the device before executing the test.
