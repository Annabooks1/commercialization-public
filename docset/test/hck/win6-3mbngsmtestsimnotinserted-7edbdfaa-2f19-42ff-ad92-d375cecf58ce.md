# Win6\_3.MBN.GSM.TestSimNotInserted


This test validates the return values of various control operations when there is no SIM inserted.

Before you run the test, complete the test setup as described in the test requirements: [Mobile Broadband Testing Prerequisites](mobile-broadband-testing-prerequisites.md).

You must remove the SIM from the device before running the test.
