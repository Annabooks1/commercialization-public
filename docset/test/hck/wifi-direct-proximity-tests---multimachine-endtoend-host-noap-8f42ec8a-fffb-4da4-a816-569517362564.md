# WiFi Direct Proximity Tests - MultiMachine EndToEnd Host NoAP


This test verifies a basic end-to-end scenario by using the PeerFinder APIs over Wi-Fi Direct. A connection between three machines is established and a small amount of data is exchanged between the host and clients.

Before you run the test, complete the test setup as described in the test requirements: [Wireless LAN (802.11) Testing Prerequisites](wireless-lan--80211--testing-prerequisites.md).

This test requires three computers.
