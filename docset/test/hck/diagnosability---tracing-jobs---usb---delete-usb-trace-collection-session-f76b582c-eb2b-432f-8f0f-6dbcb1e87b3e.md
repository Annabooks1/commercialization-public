# Diagnosability - Tracing Jobs - USB - Delete USB trace collection session


This optional test is provided to help troubleshoot certain test failures and is not required for certification.

You can use this optional test to delete a USB trace collection session. See also [Diagnosability - Tracing Jobs - USB - Start USB trace collection](diagnosability---tracing-jobs---usb---start-usb-trace-collection-f246772d-dccb-4d1e-9b68-9f823af0f456.md).
