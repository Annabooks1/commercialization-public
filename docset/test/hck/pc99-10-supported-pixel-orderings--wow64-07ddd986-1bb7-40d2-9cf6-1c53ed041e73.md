# PC99 \#10 supported pixel orderings (WoW64)


This automated test checks for supported pixel orderings.

Video cards or chipset drivers must support the following pixel orderings for their various display modes:

-   15-bit RGB 555

-   16-bit RGB 565

-   24-bit RGB 888

-   32-bit ARGB 8888
