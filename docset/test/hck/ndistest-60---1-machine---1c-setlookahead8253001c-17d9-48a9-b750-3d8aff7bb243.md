# NDISTest 6.0 - 1 Machine - 1c\_SetLookahead


This test verifies that the look ahead length on your card can be set to everything up to (but NOT exceeding) the value returned from OID\_GEN\_MAXIMUM\_LOOKAHEAD.

**Note**  
This test does not verify that the card is actually using the value correctly.
