# Gamma Correction


This automated test verifies D3D9 SRGB Texture Read/Write features (specifically the Sampler state: D3DSAMP\_SRGBTEXTUREand the Render State: D3DRS\_SRGBWRITEENABLE).

This topic applies to the following test jobs:

-   Gamma Correction

-   Gamma Correction (WoW64)
