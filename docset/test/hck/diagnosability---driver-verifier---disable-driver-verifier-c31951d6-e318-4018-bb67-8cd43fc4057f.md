# Diagnosability - Driver Verifier - Disable Driver Verifier


This optional test is provided to help troubleshoot certain test failures and is not required for certification.

This optional test can be used to disable [Driver Verifier](http://go.microsoft.com/fwlink/p/?linkid=303808).
