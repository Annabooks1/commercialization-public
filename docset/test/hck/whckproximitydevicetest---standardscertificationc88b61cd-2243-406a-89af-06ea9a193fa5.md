# WHCKProximityDeviceTest - StandardsCertification


This test verifies that an implementation that uses Near Field Communication (NFC) technology adheres to the standards of the NFC Forum.

During this test you'll see the following message:

'If this hardware implementation uses NFC technology, it must have been certified by the NFC Forum.'

You must press **Y** to complete the test.
