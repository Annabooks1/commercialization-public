# WDDM LockSmith


This automated test is a basic test of user-mode display driver locking functionality.

This test verifies that user-mode display drivers correctly implement the Lock DDI. In particular, this test simultaneously locks multiple surfaces with a resource and locks surfaces with discard semantics. Display drivers must not stop responding or fail calls to the Lock DDI during this test.
