# WHCKProximityDeviceTest - NFCForumTag


This test verifies that an implementation that uses the Near Field Communication (NFC) Forum tag adheres to the N-Mark Trademark Usage Guidelines for Tags and Media.

During this test you will see the following message:

'Devices using only an NFC Forum Tag must adhere to the N-Mark Trademark Usage Guidelines for Tags and Media'

You must press **Y** to complete the test.
