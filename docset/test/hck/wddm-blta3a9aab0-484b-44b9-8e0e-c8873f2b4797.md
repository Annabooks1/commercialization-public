# WDDM Blt


This automated test is a stability test for the user-mode Device Driver Interface (DDI) for bit-block transfer (bitblt).

This test verifies system stability when the system is performing valid surface bitblt operations for video and system memory. The test performs surface bitblts between a variety of surface types and memory pools. Longhorn Display Driver Model (LDDM) drivers must not crash in user mode or kernel mode during the test.

This topic applies to the following test jobs:

-   WDDM Blt

-   WDDM Blt (WoW64)
