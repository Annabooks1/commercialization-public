# Fidelity Test (System, Manual)


This manual test verifies hardware audio fidelity by using an Audio Precision (AP) System Two or AP2700 series analog analyzer and analog generator.

**Note**  
This is a system-specific implementation of an existing test. For more information on this test, see [Fidelity Test (Manual) - Certification](fidelity-test--manual----certification8b8c3865-bbeb-4fee-b0af-cac958abff0b.md).
