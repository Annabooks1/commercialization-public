## Running the test


Before you run the test, complete the test setup as described in the test requirements: [Mouse or other Pointing Device Testing Prerequisites](mouse-or-other-pointing-device-testing-prerequisites.md).

1.  Turn off test signing by running **bcdedit /set testsigning off** from an elevated command prompt and then restarting the computer.

2.  Start the test.

3.  Move your finger on touchpad and verify the following:

    -   The mouse cursor moves as expected.

    -   No PTP is reported. The test should show no contact visualization on the screen, and clicking the pad should not turn the bar green at the bottom of the screen.

4.  If the above conditions are met, press **P** to manually pass the test. Otherwise, press **F** to fail the test.

5.  Turn on test signing by running **bcdedit /set testsigning on** from an elevated command prompt and then restarting the computer.
