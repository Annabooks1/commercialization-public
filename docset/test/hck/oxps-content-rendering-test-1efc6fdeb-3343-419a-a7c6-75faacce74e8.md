# OXPS Content Rendering Test 1


This is a version of the XPS Content Rendering Test 1 test that is designed to validate printer capabilities for Open XML Paper Specification (XPS) documents.

This test will send OXPS files to the specific printer using XpsDocumentWrite.Write(filename) API. The test will fail if any exception is detected.

For information about this manual test, see [XPS Content Rendering Test 1](xps-content-rendering-test-1ac4cd4f7-d06b-4e7e-bae2-f2a79168d4f4.md).
