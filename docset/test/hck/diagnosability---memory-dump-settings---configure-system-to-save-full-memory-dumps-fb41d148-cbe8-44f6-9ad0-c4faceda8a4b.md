# Diagnosability - Memory Dump Settings - Configure system to save full memory dumps


This optional test is provided to help troubleshoot certain test failures and is not required for certification.

You can use this optional test to configure the system to save full memory dumps on system crashes. For more information, see [Overview of memory dump file options](http://go.microsoft.com/fwlink/p/?linkid=303767).
