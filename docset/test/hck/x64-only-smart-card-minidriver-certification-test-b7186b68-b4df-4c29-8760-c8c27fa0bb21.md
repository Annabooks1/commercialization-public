# x64 only Smart Card Minidriver Certification Test


This is a version of an existing test. This automated test validates 32-bit versions of the smartcard minidriver DLL on 64 bit computers.

For information about this test, see [Smart Card Minidriver Certification Test](smart-card-minidriver-certification-test-b3e342d5-3ae4-47d8-a13a-880a038bf5e1.md).
