# WDDM Blt - Depth


This automated test verifies that drivers are stable when they perform bit-block transfers (bitblts) to and from system memory surfaces that have depth/stencil formats.

In particular, this test creates a video memory depth buffer of a supported format. It then creates a system memory lockable depth buffer and instructs the driver to perform a bitblt between them. The driver must not crash or hang during the test.

This topic applies to the following test jobs:

-   WDDM Blt - Depth

-   WDDM Blt - Depth (WoW64)
