# WiFi Direct Scenario Tests - Legacy Clients


This automated test verifies that a Wi-Fi Direct (WFD) Group Owner (GO) that is running in legacy mode can be connected to from a legacy STA.

## Running the test


Before you run the test, complete the test setup as described in the test requirements: [Wireless LAN (802.11) Testing Prerequisites](wireless-lan--80211--testing-prerequisites.md).

In this test, two clients, the system under test (SUT) STA and the access point (AP) STA, connect to the GO.
