# DF - Fuzz zero length buffer FSCTL test (Certification)


This test issues a series of calls to the DeviceIOControl with input and output buffer lengths of 0. The test generates varying file system control codes by using different function codes, device types, data transfer methods, and access requirements.

For information included in the Windows Driver Kit on this test, see [Penetration Tests (Device Fundamentals) (Windows Drivers)](http://msdn.microsoft.com/library/windows/hardware/jj673014.aspx).
